# TicketLeap



## Commands

- Create CRUD

```
npx nx g @nrwl/nest:resource --project api --directory="app"
```

- Generate migration files

```
npm run make:migration -- --name=<name>
```

- Run migration

```
npm run migrate:run
```

- Revert migration

```
npm run migrate:revert
```

- Database seeding

```
npm run make:seed
```
